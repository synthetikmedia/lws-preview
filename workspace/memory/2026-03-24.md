### How to start voice system (2 processes in WSL2):
```bash
# Terminal 1: llama.cpp server
source /home/apps/Orpheus-FastAPI/venv/bin/activate
python3 -m llama_cpp.server --model /home/apps/models/Orpheus-3b-FT-Q4_K_M.gguf --n_ctx 2048 --host 0.0.0.0 --port 1234 --n_gpu_layers 0

# Terminal 2: Orpheus-FastAPI
cd /home/apps/Orpheus-FastAPI
source venv/bin/activate
python app.py
```
